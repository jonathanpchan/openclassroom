Developer Guide

What is Open Classroom?

As students, we have personally had issues finding open classrooms to study in. To add insult to injury, physical classroom listings are either out of date or inaccurate. Our goal is to provide students an application where they can find which classrooms and labs are open.

Currently, students have no easy way to find this out. They must chance upon a room or search physically until they find one. We want to provide a listing of what rooms are open to make the lives of students easier. In addition, students will be able to upload their schedules and the system will suggest where they will be able to study. There will be social elements where students can find other people to study with, whether they are in the same class or taking a different section. Chat functionality and note sharing will also be implemented. Furthermore, there will be a way for users to update desired classroom features such as whiteboards or outlets. This application will provide immense benefit to students because they will no longer have to stress, waste time, or debate whether said classroom is open or not.

Setup

Installing the Project and its Dependencies
  1. Download and install NodeJS
  2. Download and install MongoDB
  3. Add NodeJS, MongoDB, and NPM to your PATH
    1. In the start menu, type in "environment" and click on the item labeled "Edit environment variables for your account"
    2. Highlight PATH and click "Edit"
    3. Input the items into your PATH
  4. Download / Fork the project onto your computer
  5. In the command prompt, input the command: npm install -g nodemon
  6. Navigate to the "openclassroom" folder and input the command: npm install
  7. Navigate to the "openclassroom/angular-src" folder and input the command: npm install
Initialize Your MongoDB Database
  1. In the command prommpt, input the command: mongod
  2. In another command prompt, navigate to the "openclassroom/scraper" folder and input the command: node scraper
  3. Input the command: node StudyBuddyTesting 1
Run Your Open Classroom Application on Localhost
  1. Make sure mongod is running on one of your command prompts
  2. In another command prompt, navigate to the "openclassroom" folder and input the command: nodemon
  3. In a third command prompt, navigate to the "openclassroom/angular-src" folder and input the command: ng serve
  4. Open a web browser and input localhost:4200 in the address bar
Adding Dummmy Users into the Study Buddy
  1. Make sure your application is running on Localhost (if not, see above)
  2. In a command prompt, navigate to the "openclassroom/scraper" folder and input the command: mongoimport --db open-classroom --collection users --file dummyUsers.json
  3. In the same folder, input the command: node StudyBuddyTesting
    (Note: This command will be used if the cronjob is not up and running (See below). Currently this function does not end, so once the text stops running, terminate the program using CTRL + C)
  4. In the same folder, input the command: node studdybuddychron
    (Note: This program will be leveraged as a repeatable task. Until done so, we manually run it while the application is running. Currently this function does not end, so once the text stops running, terminate the program using CTRL + C)

Dependencies

Open Classroom uses the MEAN stack
  • MongoDB
  • Angular 2.0
  • ExpressJS
  • NodeJS

(Angular 2.0 and ExpressJS are installed using "ng install". MongoDB and NodeJS shall be installed from their respective websites.)
